# Neo4flix Frontend

## Angular Application for Movie Recommendations

### Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

### Features

- User Authentication (Login/Register)
- Movie Browsing and Search
- Movie Rating System
- Personalized Recommendations
- Responsive Design

### Technology Stack

- Angular 18+
- TypeScript
- SCSS
- Angular Material (Optional)
